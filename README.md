# Data-Analysis-Project-E-commerce-Dataset-SQL-Pandas-
Analyzed e-commerce transactional data using SQL and Pandas. Cleaned and transformed data, calculated revenue metrics (total, average per customer), identified high-value customers, and evaluated revenue distribution across products using aggregations and window functions.

Repository contains 3 files:
1. data_small - shortened variant (100r rows) of e-commerce data from kaggle (link: https://www.kaggle.com/datasets/carrie1/ecommerce-data?resource=download)
2. e-commerce data - ipynb file with code written in python pandas with basic analyse of data
3. analysis.sql - SQL script with the same analysis and additional queries
4. 3 images of graphs demonstrating sales trends
